# 光谱后端管理系统-前端内容

## 0 为啥搞这个

最近多年工作中,基本还是用不到微服务等高端的技术

但是单体的后端SpringBoot加上前端VUE框架的前后端分离的项目

倒是一直再用.为了简化开发,也简化工作的重复内容.

所以想要做一个比较通用的前后端都包含的框架

[后端位置](https://github.com/yangxj96/spectra-api)
